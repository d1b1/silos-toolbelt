Silos Toolbelt
==============
This is a proof of concept that provides a CLI for the silos framework. 

    npm install d1b1/silos-toolbelt

### Usage
For development:
  
    node index.js --list
    node index.js --create [silo]

### Goal

    silos list
    silos create [silo name]
